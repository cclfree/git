미리 읽기

git이란 버전관리를 자동적으로 해주는 툴이다.명령어로는 다음과 같다.
저장소 생성git init:저장소를 생성
커밋git add ‘파일 명’:파일이 추적이 가능하면 추적을 하고 아니면 에러 메세지를 띄움 (여기서 추적을하는 의미는 커밋을 하기위해 추적을 해야하기 때문이다.)git commit -m ‘버전 설명’:커밋(버전에 등록)
git add와 git git commit을 한번에 묶어서 하는 방법은 git commit -am ‘버전 설명’ 이다.
변경 내용git log:변경내용 확인git log 버전 아이디: 그 아이디 의 로그
