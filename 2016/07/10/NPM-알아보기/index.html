NPM의 약자는 node package manager로 node.js를 사용하는 사람들은 거의 필수로 사용함
Node.js 의 프로젝트는 의존하는 다른 패키지의 소스코드를 포함하지 않는다. 대신 의존하는 패키지들을 package.json 에 명시하고 개발자들은 npm 을 사용해 각자 설치해 사용한다.
