did not match any file(s) known to git.
1. 변경 전 파일이름을 git에서 un-add 한다.
1git reset HEAD oldFileName.filecs
2. 새로 변경 된 파일이름을 git에 add 한다.
1git add newFileName.filecs
3.다시 commit
